
Nice Trader - Buildozer package (for app.buildozer.io)

Steps to build APK on your phone:
1. Go to https://app.buildozer.io
2. Tap 'Upload project' and select the NiceTrader_Buildozer.zip file.
3. Wait for the build to complete (may take several minutes).
4. Download the produced APK and install on your phone.

First Launch:
- You will be prompted to set a 4-6 digit PIN.
- By default the app will "remember" your PIN and unlock straight to the menu on subsequent opens.
- You can change the PIN or toggle remember in Settings.
